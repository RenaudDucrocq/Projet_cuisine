# Projet Python

Ce projet est développé en Python.

## Installation

1. Clonez le dépôt :
   ```
   git clone <url-du-dépôt>
   ```
2. Installez les dépendances :
   ```
   pip install -r requirements.txt
   ```

## Utilisation

Lancez le programme principal :
```
python main.py
```

## Tests

Pour exécuter les tests :
```
pytest
```
